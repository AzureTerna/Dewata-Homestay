# Dewata Homestay Project Notes

## Project Overview
This project is a website for "Dewata Homestay" in Bali. It features an Oriental theme (Red/Gold colors) and includes sections for Home, History & Activities, and Reservation.

## Color Palette
- Primary: #8B0000 (Deep Red)
- Secondary: #D4AF37 (Gold)
- Background: #FFF8E7 (Warm White)

## Features
1. **Home Page**: Hero section, Location preview, Room packages teaser.
2. **Experience Page**: History of the homestay, List of new activities (Welcome Drink, Bike Tour, etc.).
3. **Reservation Page**: Detailed room packages, WhatsApp Booking Engine form.
4. **Global**: Quick Booking Section available near the footer on all pages.
